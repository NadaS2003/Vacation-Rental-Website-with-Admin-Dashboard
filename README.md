<h1 align="center">Vacation Rental Website with Admin Dashboard</h1>

<p align="center">
A modern hotel and vacation rental management system built with Laravel. The platform includes a responsive admin dashboard that allows administrators to manage hotels, rooms, reservations, and system users through an organized and user-friendly interface.

The system is designed to simplify hotel management operations and provide an efficient way to handle bookings and property information.
</p>

---

## 📌 Project Overview

Managing hotels and vacation rentals manually can be complex and time-consuming. This system provides a centralized platform that allows administrators to manage hotels, rooms, and reservations efficiently while maintaining an organized and responsive interface.

The system can be used as a foundation for hotel booking platforms, vacation rental systems, or property management dashboards.

---

## 🚀 Key Features

- Admin dashboard for system management
- Hotel management (Create, Read, Update, Delete)
- Room management with images and detailed information
- Reservation management and status updates
- Authentication system for administrators
- CRUD operations for core system entities
- Responsive and user-friendly interface
- Organized database structure for property management

---

## 🛠️ Tech Stack

- PHP
- Laravel 12
- Blade Templating Engine
- HTML5
- CSS3
- Bootstrap
- JavaScript
- MySQL / SQLite
